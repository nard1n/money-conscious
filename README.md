## Installation
To install, simply clone or download the repository. There are some NPM dependencies to keep in mind. Be sure to run the following:
-- npm init
-- npm i
-- npm i mongoose
-- npm i express
-- npm i lite-server
-- npm i compression
-- npm i morgan (optional)
